# Documentação do Backend

<img width="1284" height="602" alt="swagger" src="https://github.com/user-attachments/assets/75054d00-e349-40f6-81cb-51a7eaa2efba" />

O sistema de backend produzido conta com uma stack de ferramentas avançadas, permitindo a criação de uma aplicação de micro serviço simples e com o mínimo de vulnerabilidade possível:
- Docker: sistema de criação e gestão de containers de aplicações;
- Docker Compose: sistema de gestão de múltiplos contêineres, permitindo uma rápida definição e ativação da aplicação;
- Nest.js: framework backend que adota as melhores práticas de mercado por padrão, como o IoC e inversão e injeção de dependência;
- PostgreSQL: para a manipulação dos dados;
- Swagger: para a visualização da documentação da API.
